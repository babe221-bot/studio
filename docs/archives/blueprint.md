# **App Name**: Stone Lab Configurator

## Core Features:

- Dimension Input: Allows the user to set the dimensions of the stone piece (length, width, height).
- Material Selection: Allows the user to select a material (e.g., granite, marble) for the stone piece.
- Surface and Edge Configuration: Allows the user to define the surface finish (e.g., polished, matte) and edge profile (e.g., beveled, rounded) for the stone piece.
- 3D Visualization: Visualizes the configured stone piece in 3D, allowing the user to rotate and inspect the design.
- Real-time Calculation: Calculates the surface area, weight, material cost, and processing cost of the configured stone piece.
- PDF Export: Saves stone piece configuration to a work order that is stored temporarily (for the duration of the browser session), and allows generating the work order in PDF format.

## Style Guidelines:

- Primary color: Deep blue (#3B5998), reminiscent of Lapis Lazuli.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop for the app's content.
- Accent color: Warm gold (#FFD700), for highlighting key elements such as interactive buttons, enhancing user experience.
- Font: 'Inter', a sans-serif font, for clear and modern headlines and body text.
- Code Font: 'Source Code Pro', a monospaced font, to ensure code readability is not compromised and presented professionally
- Grid-based layout for arranging input fields, selections, and the 3D visualization.
- Smooth transitions and animations when updating the 3D visualization based on user inputs.
