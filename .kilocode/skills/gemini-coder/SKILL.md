# Gemini Coder

This skill provides specialized assistance for coding tasks using Gemini models.

## Usage

Use this skill to:
1.  **Refactor Code:** Ask Gemini to refactor specific functions or files.
2.  **Generate Tests:** Request unit tests for your code.
3.  **Explain Code:** Get explanations for complex logic.
4.  **Debug:** Provide error messages and code snippets for debugging.

## Guidelines

-   **Context is Key:** Always provide relevant file content when asking for changes.
-   **Review Suggestions:** Carefully review code generated by the model before applying it.
-   **Iterate:** If the initial result isn't perfect, provide feedback and ask for adjustments.

## Tools

This skill works best with the following MCP servers (configured in `.kilocode/mcp.json`):
-   `filesystem`: For file access.
-   `git`: For version control context.
-   `fetch`: For retrieving external documentation.
